# tao-skill：涛哥创业思维框架集

> 用涛哥的思维方式帮你诊断创业路上的具体问题。

## 包含 Skill

| Skill | 功能 | 触发词 |
|---|---|---|
| `tao` | 中台调度——分析你的问题涉及哪些维度，自动调度对应子 skill | `/tao`、`涛哥怎么看`、`帮我看看` |
| `tao-traffic` | 流量思维——怎么做流量、从哪来、怎么变现 | `/tao-traffic`、`怎么做流量`、`怎么引流` |
| `tao-content` | 内容思维——怎么做内容、怎么选题、怎么传播 | `/tao-content`、`怎么做内容`、`怎么选题` |
| `tao-sell` | 销售思维——怎么卖、怎么成交、怎么转化 | `/tao-sell`、`怎么做销售`、`怎么成交` |
| `tao-startup` | 创业思维——选什么赛道、怎么开始、模式怎么跑通 | `/tao-startup`、`做什么好`、`选什么赛道` |
| `tao-growth` | 成长思维——怎么提升自己、精力管理、对抗焦虑 | `/tao-growth`、`怎么提升自己`、`时间不够用` |
| `tao-network` | 人脉思维——怎么混圈、怎么认识牛人、让人脉变资产 | `/tao-network`、`怎么混圈`、`怎么认识牛人` |

## 核心设计

**不是搜索引擎，是诊断工具。**

每个 skill 都不返回「涛哥说过 XX」，而是用涛哥的思维方式帮你诊断问题、给处方、定行动。

**追问机制：** 回答前，先通过1-5轮追问（一次一个问题）把用户的真实需求带出来，等 AI 有 95% 信心理解了真实目标后，才给出方案。

## 安装

把 `skills/` 下的所有目录复制到你的 `.claude/skills/` 目录即可：

```bash
cp -r skills/* ~/.claude/skills/
# 或项目级
cp -r skills/* /你的项目/.claude/skills/
```

## 使用

直接说你的问题，中台 `tao` 会自动分析维度并调度对应的子 skill。

也可以直接调用某个维度的 skill，如 `/tao-traffic 我的抖音流量很差怎么办`。
