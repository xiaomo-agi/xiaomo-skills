## 他是谁

**杨涛**，江湖人称赛博菩萨、互联网百晓生、创业云暖男。

二十年连续创业者。公众号《屋里涛说》，知识星球《平民创业手册》，《确定性成长》作者。

他最大的特点是 **「实战派 + 大白话」**——不是"教别人创业"的人，他自己一直在创业，分享的全是亲身踩过的坑和验证过的打法。同样的"流量获取""精细化运营"，他能用大白话把晦涩道理讲透，让完全没基础的人听完直接上手实操。

平均每年回答数千位创业者的各类问题，包括但不限于职场、互联网、副业、创业、流量、产品、运营、销售、团队管理、个人成长，以及各种黑科技和方法论。

**这份 Skill 基于他公开输出的 970 万字蒸馏而成。**

但蒸馏永远是冷的，代替不了和他本人沟通交流的那种「本真鲜活，能量涌动」。真心希望大家能和他做朋友，人间值得。

## 这份 Skill 能做什么

**不是搜索引擎，是诊断工具。**

你不问"涛哥说过什么"，而是让涛哥帮你诊断具体问题——卡在哪、为什么、下一步做什么、做到什么标准。

### 包含 7 个思维框架

| Skill | 功能 | 触发词 |
|---|---|---|
| `tao` | 中台调度——分析你的问题涉及哪些维度，自动调度对应子 skill | `/tao`、`涛哥怎么看`、`帮我看看` |
| `tao-traffic` | 流量思维——怎么做流量、从哪来、怎么变现 | `/tao-traffic`、`怎么做流量`、`怎么引流` |
| `tao-content` | 内容思维——怎么做内容、怎么选题、怎么传播 | `/tao-content`、`怎么做内容`、`怎么选题` |
| `tao-sell` | 销售思维——怎么卖、怎么成交、怎么转化 | `/tao-sell`、`怎么做销售`、`怎么成交` |
| `tao-startup` | 创业思维——选什么赛道、怎么开始、模式怎么跑通 | `/tao-startup`、`做什么好`、`选什么赛道` |
| `tao-growth` | 成长思维——怎么提升自己、精力管理、对抗焦虑 | `/tao-growth`、`怎么提升自己`、`时间不够用` |
| `tao-network` | 人脉思维——怎么混圈、怎么认识牛人、让人脉变资产 | `/tao-network`、`怎么混圈`、`怎么认识牛人` |

### 追问机制

回答前，先通过 1-5 轮追问（一次一个问题）把你的真实需求挖出来。等 AI 有 95% 信心理解了你的真实目标后，才给出方案。

## 安装

把 `skills/` 下的所有目录复制到你的 `.claude/skills/` 目录即可：

```bash
# macOS / Linux
cp -r skills/* ~/.claude/skills/

# Windows
xcopy /E /I skills\* %USERPROFILE%\.claude\skills\

# 或项目级（推荐）
cp -r skills/* /你的项目/.claude/skills/
```

## 使用

直接说你的问题，中台 `tao` 会自动分析维度并调度对应的子 skill。

也可以直接调用某个维度的 skill：

```
/tao-traffic 我的抖音流量很差怎么办
```
